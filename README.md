# Vanta.js Animation Website

An interactive website built to explore and experiment with beautiful animated backgrounds using Vanta.js.  
This project demonstrates how Vanta effects can enhance modern web designs with dynamic visuals.

## Live Link

https://maheshkn30.github.io/Vanta-animation-website/

## Usage

- Click the effect buttons in the header to switch animations
- Each effect creates a unique interactive background
- Use this project as a reference for integrating Vanta.js into your own websites

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Vanta.js
- Three.js
